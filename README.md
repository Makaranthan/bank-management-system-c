# Bank Management System (C)

## Description
A simple CLI-based bank management system using C to handle account creation, deposits, withdrawals, and balance checks.

## Features
- Create new account
- Deposit money
- Withdraw money
- Check account balance
- Menu-driven using switch-case and functions

## Technologies Used
- C programming
- Structs, Functions, Loops, Conditional Statements

## How to Run
Compile and run using any C compiler:gcc bank_management.c -o bank
./bank


## Learning Outcome
- Practiced core C programming concepts
- Implemented functions and structs
- Developed logical thinking and problem-solving skills
